# АК: тренажёр билетов

Локальный тренажёр карточек для подготовки к экзамену по архитектуре компьютера.

## Что внутри

- `index.html` - автономный тренажёр карточек.
- `ak_exam_all_cards.csv` - полная CSV-колода на 233 вопроса.

## Как пользоваться

Открой `index.html` в браузере. Полная колода уже встроена в страницу, но при необходимости можно загрузить CSV вручную.

Горячие клавиши:

- `Space` - перевернуть карточку.
- `ArrowRight` - следующая карточка.
- `ArrowLeft` - предыдущая карточка.
- `X` - отметить "попал".
- `Z` - отметить "не попал".

На мобильном экране кнопки закреплены снизу, а карточка прокручивается внутри.
